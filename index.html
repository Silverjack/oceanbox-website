Shipping Container Types

Mainstream ISO Container Types for North America Operations

Built around widely used shipping-container classifications, this range helps you match cargo profile, handling method, and delivery constraints with the right equipment type.

Dry Containers

20GP / 40GP / 40HC

  • Standard choice for most general cargo
  • Available in cargo-worthy and one-trip conditions
  • Suitable for storage, repositioning, and inland delivery

High Cube

40HC / 45HC

  • Extra internal height for volume-sensitive cargo
  • Common for distribution, retail stock, and project loads
  • Well suited to high-cube warehouse interfaces

Special Door Configurations

Side-Open / Double-Door / Duocon

  • Faster loading and unloading for oversized cargo
  • Ideal for conversion, events, and technical projects
  • Improves access when fork-lift side entry is required

Reefer Containers

20RF / 40RF High Cube

  • Temperature-controlled transport and storage
  • Used for cold chain, food, and sensitive materials
  • Available for short-term and long-term programs

Open Top

20OT / 40OT

  • Top-loading design for machinery and tall cargo
  • Flexible handling for crane-loaded shipments
  • Common in industrial and project-cargo lanes

Flat Rack

20FR / 40FR

  • Designed for oversized and out-of-gauge cargo
  • Strong lashing structure for heavy project equipment
  • Preferred for machinery, steel, and long cargo pieces

Tank Containers

20TK / Special Liquid Cargo

  • For bulk liquid and chemical cargo movements
  • Configuration depends on product type and compliance rules
  • Supplied by request with route-specific planning support

Modified and Production Orders

Custom Build and Factory Program

  • From spec definition to production and delivery
  • Supports project modules and specialized configurations
  • Single accountable execution from origin to destination
